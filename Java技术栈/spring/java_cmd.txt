cmd (win + R)
1.切换盘符   盘符：（不区分大小写）
2.进入文件夹   cd 文件夹名
  进入多级文件夹  cd\文件夹1\文件夹2
  返回上一级   cd ..
   返回根路径   cd \
3.查看当前内容   dir(directory)
4.清屏   cls  (clear screen)
5.退出   exit
6.tab   补全
7. 上下 上下条命令


IDEA快捷键
alt+enter   错误解决
ctr+d   删除所在行
shift+F6   批量改方法变量
ctr+/   单行注释
ctr+alt+/   多行注释
ctl+alt+l   代码格式化
ctr+单击   进入定义处
