# node.js 更新

使用n管理工具更新

```text
npm install -g n
```

升级node.js到最稳定版本

```text
n stable
```

升级node.js到最新版本

```text
n latest
```

升级node.js到指定版本

```text
n v14.15.0
```

切换node.js的使用版本

```text
n v12.16.1 (ENTER)
```

删除指定版本

```text
n rm v12.16.1
```

用定制的版本执行脚本

```text
n use 12.16.1 test.js
```

更新npm版本

```text
npm install -g npm
```
