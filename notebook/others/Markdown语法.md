# Markdown 语法

常用的语法已经记住了，但是还需要学习一些新的语法

## 页内跳转

主要有两种方式：markdown方式和html方式

### markdown方式

和插入链接是类似的，`[xxx](url)`是链接的语法，不仅可以用来跳转页外，还可以用来业内跳转

语法: `[xxx](#xxx)`xxx一般是标题，且是唯一的

只能跳转到标题，不够灵活，但是够用啦

### html方式

利用html的方式定义一个锚点，然后使用跳转语法

锚点定义：`<span id='jump'>跳转到这里</span>`

使用：`[点击跳转](#jump)`，id是唯一的

或者：`<a href='#jump'>点击跳转</a>`

**说明：ctr + 左键 才能跳转**
